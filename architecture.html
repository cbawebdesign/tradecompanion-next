CORS Proxy (localhost only)

On localhost, proxyUrl() rewrites absolute Azure/Scanzzers URLs to relative paths.
Next.js rewrites in next.config.js forward them to the real endpoints.
In production, URLs pass through unchanged — no proxy needed.

04 Hooks Reference
Hook Type Endpoint / URL Interval Dedup Strategy Init Delay
useSignalR WebSocket /api/negotiate (Azure) or direct localhost:5000/lxhub Persistent. Token refresh: 40min. Stale re-sub: 30s check / 3min threshold. Batch: 250ms Per-event (see SignalR events table) —
useNewsHub WebSocket stage.news.scanzzers.com/newshub
JWT via POST /auth/machine-login		 Persistent. Reconnect check: 30s. JWT refresh on reconnect Same as addAlert() dedup (symbol+msg+type) —
useTweetsPolling REST Poll GET /api/tweets?since=${id_long} 30s Track id_long in seenTweetIdsRef. Initial: limit 100 500ms
useFilingsPolling REST Poll GET /api/Filings?since=${date} 60s By dcn or ${symbol}-${form}-${time_et}. Filters to watchlist symbols 2s
useCatalystPolling REST Poll GET /api/Catalyst?since=${date} 60s By ${symbol}-${saveTime_et}. Filters to watchlist symbols (high volume) 5s
useTradeExchangePolling REST Poll GET /api/TradeExchangeGet?since=${date} 30s By id. Parses $CASHTAG + leading ticker from content 3.5s
usePrevCloses REST GET /api/PrevCloses 60min N/A — overwrites. Seeds quotes for symbols without live data —
usePriceAlerts Local — On quote update (reactive) Key: upper-${symbol}-${price} or lower-... —
useCrossWindowSync Local localStorage trade-companion-storage On storage event Deep equality (JSON.stringify) skip if same —
useQuoteBroadcaster Local BroadcastChannel trade-companion-quotes 500ms throttle N/A — latest state wins —

06 SignalR Event Reference

Broadcast1sBar S→C [{s, c (close), v}] batchQuoteUpdate 1-second candlestick bars
BroadcastAlertTrigger S→C {time, alert, alertName, watchlist, message, symbol, color} addAlert Legacy format. Parses alert JSON for url
newFiling S→C {symbol, alert, message, title, form_type, url, Url} addAlert Color ■ #00bcd4
newTradeExchange S→C {data} addAlert Color ■ #ff9800
newCatalystScanner S→C {data} addAlert Color ■ #9c27b0
BroadcastNews S→C {symbol, headline, story_id, savetime_et, time_et} addAlert Color ■ #7c4dff. URL: ampspr?id=${story_id}
tradingViewAlertRaw S→C {id, raw_text, received_utc} addAlert Color ■ #4caf50. Dedup by id. Backfill on start
ScannerAlert S→C {symbol, pctChange, price, prevClose, session, bucket} addScannerAlert Goes to scannerAlerts[], NOT alerts[]
SubL1 C→S symbol: string — Subscribe to quotes. 150ms delay between each invoke

◆ proxyUrl() (lib/proxyUrl.ts)

Active only on localhost (checks window.location.hostname).
Maps full URLs to proxy paths: https://tradecompanion3.azurewebsites.net/api/foo → /tc3/api/foo
In production, returns the URL unchanged — browser calls the external service directly.

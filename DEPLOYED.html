New Cosmos DB Containers

AlertLog

Every alert the web app fires gets stored here. Partition key: /date

  • 7-day TTL (auto-cleanup)
  • Fields: symbol, type, message, timestamp, receivedAt, source, hookName
  • Written by web app via POST /tcadmin/alert-log
  • Read by AlertAuditTimer for integrity comparison

AlertAuditReports

Daily integrity reports generated by the timer agent. Partition key: /date

  • 30-day TTL (auto-cleanup)
  • Fields: date, score (%), backendTotal, webTotal, matched, missed, delayed, duplicate
  • Includes list of missed alert details
  • Read via GET /tcadmin/audit-report

Performance Fixes Deployed

Removed Quotes from Persist

Was JSON-serializing hundreds of symbols to localStorage every 250ms. Main thread blocked, UI froze.

Hook Isolation

Split 13 hooks from 1 component into 4 isolated groups. Quote-frequency re-renders no longer cascade to all hooks.

Price Alerts Rewrite

Moved from React useEffect (fired every 250ms) to Zustand subscribe() outside React render cycle. Only checks symbols with alerts set.
